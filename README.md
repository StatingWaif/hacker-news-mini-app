## Запуск приложения

Для запуска выполните следующие шаги:

1. Склонируйте репозиторий:

   ```bash
   git clone https://github.com/StatingWaif/hacker-news-mini-app.git
   ```

2. Установите зависимости, используя Yarn:

   ```bash
   yarn install
   ```

3. Запустите приложение:

   ```bash
   yarn start
   ```
